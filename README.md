# 🏋️‍♂️ Workout API

![Status](https://img.shields.io/badge/status-active-green)
![Python](https://img.shields.io/badge/Python-3.12-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-Framework-brightgreen)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

API REST desenvolvida com **FastAPI**, **SQLAlchemy Async**, **PostgreSQL** e executada via **Docker Compose**.  
A aplicação gerencia atletas, categorias e centros de treinamento, mantendo um fluxo moderno, assíncrono, seguro e escalável.
